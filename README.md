# Mini-calc

📌 Mini Calculator (Python CLI Project)

A Python-based command-line calculator that performs basic and advanced arithmetic operations with history tracking and continuous calculations.

🚀 Features
➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division        
📊 Modulo operation
⚡ Power calculation
√ Square root support
📜 Calculation history tracking
🔁 Continuous calculations using previous result
❌ Input validation and error handling
🛠️ Technologies Used
Python 3
Built-in math and os modules
▶️ How to Run
python filename.py
